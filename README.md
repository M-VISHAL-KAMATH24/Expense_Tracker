Project Overview
This is a full-stack Expense Tracker Dashboard web application that enables users to effectively manage and analyze their income and expenses. The system supports user authentication, profile management, and interactive financial overview with advanced 3D charts for insightful data visualization.

Features
User Signup/Login with JWT authentication.

Manage profile information including editable avatar, email, job, name, and phone.

Add, edit, delete income and expenses.

Dynamic Overview with interactive charts:

3D Pie Chart for expense breakdown by category.

3D Bar Chart for monthly income vs expenses.

Radar Chart for spending patterns.

Responsive, modern UI with glassmorphism effects and smooth animations.

Real-time chatbot assistant to extract and analyze expenses and income.

MongoDB Atlas database backend with secure data storage.

Usage
Signup and login to access your dashboard.

Add/edit your profile with avatar and personal info.

Add your incomes and expenses via the dashboard.

View dynamic charts and insights tailored to your data.

Chat with SmartBot to get analysis and add expenses via chat.

Tech Stack
Backend: Node.js, Express.js, MongoDB, Mongoose, JWT

Frontend: React.js, Tailwind CSS, Recharts, Axios

Chatbot AI: Google Gemini 2.0 via FastAPI Python server

Database: MongoDB Atlas (cloud database)
